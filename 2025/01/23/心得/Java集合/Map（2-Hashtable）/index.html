  1. 判断 value 是否为 null，不为空则执行下面的操作
  2. 计算键的哈希值，通过哈希值获取在哈希桶数组的下标
  3. 将 table 数组 index 位置的值转换为 Entry 类型的数据

  5. 如果没有，则调用 addEntry 进行添加，要先判断是否需要扩容（如果哈希桶数组中 Entry 键值对的数量大于扩容阈值 threshold，则调用 rehash()进行扩容）
